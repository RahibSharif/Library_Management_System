# Library_Management_System

A Python command-line application for managing library books.

## Features

- Add books
- View books
- Search books
- Delete books
- Borrow books
- Return books
- Due dates
- Overdue book tracking
- Sort books by title
- Sort books by author
- Save and load data using JSON

## Technologies

- Python
- OOP (Classes and Objects)
- JSON
- Git & GitHub

- ## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python main.py
```
